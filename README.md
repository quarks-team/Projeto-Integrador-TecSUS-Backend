## Docker

```bash
# run mysql container
$ docker compose up mysql -d
```

## Installation

```bash
$ npm install -g yarn

$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev
```

## Test

```bash
# unit tests
$ yarn test

# test coverage
$ yarn run test:cov
```
